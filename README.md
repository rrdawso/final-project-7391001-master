# Tic-Tac-Toe Challenge
CS 1741: Web Design with JavaScript - John Bartucz

## Description
A web-based Tic-Tac-Toe game with a clean interface and singleplayer mode. The project includes a mock login/signup system (non-functional demo) and responsive design.

## Features
- Singleplayer Tic-Tac-Toe gameplay
- Responsive web design
- Navigation bar with home and game options
- Mock login/signup interface
- Mobile-friendly layout

## Project Structure
```
final-project-7391001-master/

├── game.html          # Tic-Tac-Toe game page
├── stylesheets/       # CSS styling
│   └── home.css       # Main stylesheet
├── img/               # Image assets

```

## Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Edge, etc.)
- Node.js (for development server)

## Game Features
- Classic 3x3 Tic-Tac-Toe grid
- Player vs Computer mode
- Win detection
- Demo account system
- Responsive design for all screen sizes

## Technical Implementation
- HTML5 for structure
- CSS3 for styling and responsive design
- JavaScript for game logic
- Browser-sync for development
